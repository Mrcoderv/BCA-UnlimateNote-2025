Primitive Data Types in JavaScript

Primitive data types are the most basic data types available within the JavaScript language. There are 7 types of primitive data types in JavaScript:

  1. Number: int, float, double, long, short, byte
    Example: 10, 10.5, 10e
    How to set:
    let age = 10;
  2. String: used to store text
    Example: "Hello, World!", 'JavaScript', `Template literal`
    How to set:
    let name = "John";
  3. Boolean: represents a logical entity and can have two values: true and false
    Example: true, false
    How to set:
    let isActive = true;
  4. Undefined: a variable that has not been assigned a value
    Example: undefined
    How to set:
    let x;
  5. Null: represents the intentional absence of any object value
    Example: null
    How to set:
    let y = null;
  6. Symbol: a unique and immutable primitive value and may be used as the key of an object property
    Example: Symbol('description')
    How to set:
    let sym = Symbol('unique');
  7. BigInt: represents whole numbers larger than 253 - 1
    Example: 1234567890123456789012345678901234567890n
    How to set:
    let bigIntNum = 1234567890123456789012345678901234567890n;

operator type

JavaScript operators are used to assign values, compare values, perform arithmetic operations, and more.

Arithmetic Operators

Used to perform arithmetic on numbers:

  • Addition (+): let sum = 10 + 5; // 15
  • Subtraction (-): let difference = 10 - 5; // 5
  • Multiplication (*): let product = 10 * 5; // 50
  • Division (/): let quotient = 10 / 5; // 2
  • Modulus (%): let remainder = 10 % 3; // 1
  • Exponentiation (**): let power = 10 ** 2; // 100

Non-Primitive Data Types

Non-primitive data types are more complex data types and can be used to store collections of data or more complex entities.

Type Description Example
Object Used to store collections of data and more complex entities. let person = { name: "John", age: 30 };
Array Used to store multiple values in a single variable. let colors = ["red", "green", "blue"];
Function Used to define a block of code designed to perform a particular task. function greet() { return "Hello, World!"; }
Date Used to work with dates and times. let now = new Date();
RegExp Used for matching text with a pattern. let pattern = /w+/g;
Map Used to store key-value pairs. let map = new Map(); map.set('key', 'value');

Assignment Operators

Used to assign values to variables:

  • Assignment (=): let x = 10;
  • Addition assignment (+=): x += 5; // x = x + 5
  • Subtraction assignment (-=): x -= 5; // x = x - 5
  • Multiplication assignment (*=): x *= 5; // x = x * 5
  • Division assignment (/=): x /= 5; // x = x / 5
  • Modulus assignment (%=): x %= 5; // x = x % 5
  • Exponentiation assignment (**=): x **= 2; // x = x ** 2

Comparison Operators

Used to compare two values:

  • Equal to (==): 10 == 10; // true
  • Not equal to (!=): 10 != 5; // true
  • Strict equal to (===): 10 === 10; // true
  • Strict not equal to (!==): 10 !== 5; // true
  • Greater than (>): 10 > 5; // true
  • Less than (<): 10 < 5; // false
  • Greater than or equal to (>=): 10 >= 5; // true
  • Less than or equal to (<=): 10 <= 5; // false

Logical Operators

Used to perform logical operations:

  • Logical AND (&&): true && false; // false
  • Logical OR (||): true || false; // true
  • Logical NOT (!): !true; // false

Bitwise Operators

Used to perform bitwise operations:

  • AND (&): 5 & 1; // 1
  • OR (|): 5 | 1; // 5
  • NOT (~): ~5; // -6
  • XOR (^): 5 ^ 1; // 4
  • Left shift (<<): 5 << 1; // 10
  • Right shift (>>): 5 >> 1; // 2
  • Zero fill right shift (>>>): 5 >>> 1; // 2

Other Operators

Other useful operators:

  • Conditional (ternary) operator (?:): let result = (10 > 5) ? 'Yes' : 'No'; // 'Yes'
  • Typeof operator: typeof 10; // 'number'
  • Instanceof operator: {} instanceof Object; // true

Control Flow Statements

Control flow is the order in which the computer executes statements in a script. A control flow statement is used to specify the order in which statements are executed.

If Statement

Used to execute a block of code if a specified condition is true:


if (condition) {
    // code to be executed if condition is true
}

If...Else Statement

Used to execute a block of code if a condition is true, otherwise another block of code is executed:


if (condition) {
    // code to be executed if condition is true
} else {
    // code to be executed if condition is false
}

Else If Statement

Used to specify a new condition to test if the first condition is false:


if (condition1) {
    // code to be executed if condition1 is true
} else if (condition2) {
    // code to be executed if condition1 is false and condition2 is true
} else {
    // code to be executed if both condition1 and condition2 are false
}

Switch Statement

Used to perform different actions based on different conditions:


switch (expression) {
    case value1:
        // code to be executed if expression === value1
        break;
    case value2:
        // code to be executed if expression === value2
        break;
    default:
        // code to be executed if expression doesn't match any case
}

Looping statement:

For Loop

Used to execute a block of code a number of times:


for (initialization; condition; increment) {
    // code to be executed
}

While Loop

Used to execute a block of code as long as a specified condition is true:


while (condition) {
    // code to be executed
}

 Example:

let i=0;
while(i<4){
    console.log(i);
    i++;
}

Do...While Loop

Similar to the while loop, but the block of code is executed once before the condition is tested:


do {
    // code to be executed
} while (condition);

Example:

let i=0;
do{ 
    console.log(i);
    i++;
}while(i<4);

For...In Loop

Used to loop through the properties of an object:


for (variable in object) {
    // code to be executed
}

Example:

let person = { name: "John", age: 30 };
for (let key in person) {
    console.log(key+": "+person[key]); // john, 30
}

For...Of Loop

Used to loop through the values of an iterable object:


for (variable of iterable) {
    // code to be executed
}

Example:

let colors = ["red", "green", "blue"];
console.log("Displaying color"); 
for (let color of colors) {
    console.log(color); // red, green, blue
    }

Control Flow Modifiers

Control flow modifiers alter the normal flow of execution in a script.

Break Statement

Used to terminate the current loop, switch, or label statement and transfer program control to the statement following the terminated statement:


for (let i = 0; i < 10; i++) {
    if (i === 5) {
        break; // exits the loop when i is 5
    }
    console.log(i);
}

Continue Statement

Used to skip the current iteration of a loop and continue with the next iteration:


for (let i = 0; i < 10; i++) {
    if (i === 5) {
        continue; // skips the iteration when i is 5
    }
    console.log(i);
}

Functions

Functions are blocks of code that can be defined once and executed multiple times. They can accept parameters and return values.

Defining a Function

Functions are defined using the function keyword, followed by the function name, a list of parameters in parentheses, and the code block in curly braces:


function greet() {
    return "Hello, World!";
}
console.log(greet()); // Hello, World!

Function Parameters

Functions can accept parameters, which are variables that are passed to the function when it is called:


    
Example:

function greet(name) {
    return "Hello, " + name + "!";
}
console.log(greet("John")); // Hello, John!

Arrow Functions

Arrow functions provide a more concise syntax for defining functions:


    
Example:

let greet = (name) => {
    return "Hello, " + name + "!";
};
console.log(greet("Raghav JI")); // Hello, Raghav JI!

Callback function:

A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.


    
Example:

function greet(name, callback) {
    return callback(name);
}
function sayHello(name) {
    return "Hello, " + name + "!";
}
console.log(greet("John", sayHello)); // Hello, John!

Anonymous Function

Anonymous functions are functions that are not defined with a name:


    
Example:

let greet = function(name) {
    return "Hello, " + name + "!";
};
console.log(greet("John")); // Hello, John!

Nested function

Functions can be defined inside other functions:


    
Example:

function greet(name) {
    function sayHello() {
        return "Hello, " + name + "!";
    }
    return sayHello();
}
console.log(greet("John")); // Hello, John!

Return Statement

Used to exit a function and return a value:


        
Example:

function sum(a, b) {
    return a + b; // exits the function and returns the sum
}
console.log(sum(5, 10)); // 15

Throw Statement

Used to throw an exception handeling .


        
Example:

function checkAge(age) {
    if (age < 18) {
        throw 'Underage'; // throws an exception if age is less than 18
    }
    return 'Adult';
}
try {
    console.log(checkAge(15));
} catch (e) {
    console.log(e); // Underage
}

Regular expression

Regular expressions are patterns used to match character combinations in strings. They can be used to search, edit, or manipulate text and data.

Creating a Regular Expression

Regular expressions can be created using the RegExp constructor or by using a literal syntax:


    
Example:

let pattern = /w+/g;
let pattern = new RegExp('w+', 'g');

Regular Expression Methods

Regular expressions have methods that can be used to perform operations on strings:

  • test(): Tests for a match in a string and returns true or false.
  • exec(): Searches for a match in a string and returns the result as an array.
  • match(): Searches for a match in a string and returns an array of matches.
  • search(): Searches for a match in a string and returns the index of the match.
  • replace(): Searches for a match in a string and replaces the match with a new string.
  • split(): Splits a string into an array of substrings based on a regular expression.

Regular Expression Patterns

Regular expressions can contain patterns that match specific characters or character combinations:

  • .: Matches any single character except newline characters.
  • \d: Matches any digit character.
  • \w: Matches any word character.
  • \s: Matches any whitespace character.
  • [abc]: Matches any character inside the brackets.
  • [^abc]: Matches any character not inside the brackets.
  • a|b: Matches either a or b.
  • \b: Matches a word boundary.
  • ^: Matches the beginning of a string.
  • $: Matches the end of a string.

Regular Expression Flags

Regular expressions can have flags that are used to perform different operations:

  • i: Perform case-insensitive matching.
  • g: Perform a global match (find all matches rather than stopping after the first match).
  • m: Perform multiline matching.

quantifiers

Quantifiers specify the number of occurrences of a character or group of characters:

  • *: Matches zero or more occurrences of the preceding character.
  • +: Matches one or more occurrences of the preceding character.
  • $:
  • ?: Matches zero or one occurrence of the preceding character.
  • {n}: Matches exactly n occurrences of the preceding character.
  • {n,}: Matches n or more occurrences of the preceding character.
  • {n,m}: Matches between n and m occurrences of the preceding character.

Regular Expression Examples

Examples of regular expressions:


    
Example:

let pattern = /w+/g;
let text = 'Hello, World!';
console.log(pattern.test(text)); // true
console.log(pattern.exec(text)); // ['Hello']
console.log(text.match(pattern)); // ['Hello']
console.log(text.search(pattern)); // 0
console.log(text.replace(pattern, 'Hi')); // Hi, World!
console.log(text.split(pattern)); // ['Hello', 'World!']
