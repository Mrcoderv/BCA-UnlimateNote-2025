Colors

CSS allows you to define colors using names, HEX, RGB, or HSL values.

This text is red using a color name.

This text is blue using a HEX value.

This text is tomato using RGB.

p { color: red; }
p { color: #3498db; }
p { color: rgb(255, 99, 71); }

Typography

Control font family, size, weight, style, and line height.

This is a monospace font.

This text is 24px in size.

This text is bold.

p { font-family: 'Courier New', Courier, monospace; }
p { font-size: 24px; }
p { font-weight: bold; }

Box Model

The CSS box model defines the content, padding, border, and margin of an element.

Example Box
div { width: 150px; padding: 20px; border: 5px solid black; margin: 10px; background: #f0f0f0; }
