2.1 Introduction

2.1.1 History

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of web pages using markup. HTML elements are the building blocks of HTML pages.

HTML (HyperText Markup Language) was created by Tim Berners-Lee in 1991.

Initially, HTML 1.0 was released for basic web pages.

Subsequent updates included HTML 2.0, 3.2, and 4.01 with additional features.

HTML5, released in 2014, is the latest version and supports new features like audio, video, canvas, and semantic tags.

2.1.2 Concept of Tag

HTML tags are keywords surrounded by angle brackets like <tagname>. They are used to define the structure of an HTML document.

Tags are usually paired, with an opening tag <tagname> and a closing tag </tagname>.

2.1.3 Basic Structure of HTML Document

Every HTML document should have the following structure:

        
            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Document</title>
            </head>
            <body>
                <h1>Hello, World!</h1>
            </body>
            </html>

2.1.4 HTML Elements and attributes

HTML elements are the building blocks of HTML pages. An HTML element is defined by a start tag, some content, and an end tag:

some tage are:
Tag Description
<h1> to <h6> Defines HTML headings, with <h1> being the highest (or most important) level and <h6> the least.
<p> Defines a paragraph.
<a> Defines a hyperlink, which is used to link from one page to another.
<img> Defines an image.
<ul>, <ol>, <li> Defines lists. <ul> is an unordered list, <ol> is an ordered list, and <li> defines a list item.
<table>, <tr>, <td> Defines a table. <table> is the table element, <tr> defines a table row, and <td> defines a table cell.
<form>, <input>, <textarea> Defines a form for user input. <form> is the form element, <input> defines an input field, and <textarea> defines a multi-line input field.

Attributes

Attributes provide additional information about HTML elements. They are always included in the opening tag and usually come in name/value pairs like name="value".

Some common attributes include:

Attribute Description
href Specifies the URL of a link (used in <a> tags).
src Specifies the source of an image (used in <img> tags).
alt Provides alternative text for an image (used in <img> tags).
id Specifies a unique id for an element.
class Specifies one or more class names for an element (used for CSS styling).
style Specifies inline CSS styles for an element.
title Provides additional information about an element (displayed as a tooltip).

2.2.2 Creating Hyperlinks

Hyperlinks are created using the <a> tag. The href attribute specifies the URL of the page the link goes to:

    
        <a href="https://www.example.com">Visit Example.com</a>

There are different types of links:

Type Description Example
Internal Link Links to another page within the same website. <a href="about.html">About Us</a>
External Link Links to a page on a different website. <a href="https://www.example.com">Visit Example.com</a>
Anchor Link Links to a specific section within the same page. <a href="#section1">Go to Section 1</a>

2.2.3 Images

Images in HTML are defined with the <img> tag. The src attribute specifies the URL of the image file:

    
        <img src="image.jpg" alt="Description of image">

The alt attribute provides alternative text for the image, which is displayed if the image cannot be loaded or by screen readers for visually impaired users.

2.4 Tables and Forms

2.4.1 HTML Table Elements

Tables are defined using the tag. <table>

<th> → Table header

<tr> → Table row

<td> → Table data

Merging Rows and Columns:

HTML allows you to merge cells using the colspan and rowspan attributes:

    
        <table border="1">
            <tr>
                <td colspan="2">Merged Columns</td>
            </tr>
            <tr>
                <td rowspan="2">Merged Rows</td>
                <td>Cell 1</td>
            </tr>
            <tr>
                <td>Cell 2</td>
            </tr>
        </table>

2.5 Semantic Elements

Semantic elements provide meaning to the content they enclose. They help search engines and screen readers understand the structure of a page:

  • <header> → Header section
  • <nav> → Navigation links
  • <main> → Main content
  • <article> → Article content
  • <section> → Section of content
  • <aside> → Sidebar content
  • <footer> → Footer section

2.6 Multimedia Elements

HTML5 introduced new multimedia elements for audio and video:

  • <audio> → Audio player
  • <video> → Video player

Example:

    
        <audio controls>
            <source src="audio.mp3" type="audio/mp3">
        </audio>
        <video controls width="320" height="240">
            <source src="video.mp4" type="video/mp4">
        </video>
